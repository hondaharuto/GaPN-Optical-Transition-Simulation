# GaPN-Optical-Transition-Simulation

第一原理計算ソフト「WIEN2k」と連携し、化合物半導体（GaPN混晶）における光励起状態を擬似的に生成・解析するためのシミュレーションおよびデータ可視化パイプラインです。

## 概要 (Overview)
次世代の高効率太陽電池として期待される「中間バンド型太陽電池」の材料候補であるGaPN混晶を対象に、第一原理計算を用いて二段階光吸収の励起強度依存性をシミュレーションしました。
本リポジトリでは、既存の計算ソフト（WIEN2k）には備わっていない「任意の光励起状態におけるキャリア分布の再計算」を実現するため、計算結果のk点重み係数（weight）を独自の物理モデルに基づいて操作・再投入するアルゴリズムを実装しています。

## 物理的背景 (Background)
GaPに微量の窒素（N）を添加したGaPN混晶は、N原子の局在準位とGaPの伝導帯が相互作用することで、バンドギャップ内に「バンドテイル」と呼ばれる特異な状態密度分布を形成します。このバンドテイルを中間バンド（IB）として利用し、2つの異なるエネルギーの光子を段階的に吸収させることで、太陽電池の理論変換効率の限界を突破することが期待されています。
本研究では、この二段階光吸収における「第一段階の光励起強度」が「第二段階の光吸収係数」にどのような影響を与えるかを定量的に評価しました。

## 計算モデル (Computational Model)
混晶のランダムな原子配置を厳密に評価するため、以下のスーパーセルモデルを構築しました。
* **ベース構造:** GaPの閃亜鉛鉱構造の $4 \times 4 \times 4$ スーパーセル（原子数128個）
* **N組成 (3.125%):** セル内のP原子2個をN原子に置換（$\mathrm{Ga}_{64}\mathrm{P}_{62}\mathrm{N}_2$）
* **配置パターン:** N原子の置換位置の相対距離が異なるすべての組み合わせ（全9パターン：`GaPN_01` ～ `GaPN_09`）を網羅。各配置におけるバンドギャップ変動（1.628eV ～ 1.915eV）と等価構造数を考慮し、物理的に妥当なアンサンブル平均（重み付き平均）を算出する構成としています。

## 独自アルゴリズム：光励起状態の擬似生成
第一原理計算によって得られた基底状態の電子状態データ（k点空間の各状態の占有数）に対し、以下のアルゴリズムで光励起によるキャリア再分布をシミュレートするPythonスクリプトを開発しました。

1. **キャリア抽出:** 価電子帯（VB）トップから任意のエネルギー範囲にある電子（weight）を抽出。
2. **キャリア充填:** 抽出した電子を、伝導帯（CB）下端（バンドテイル）から順に、各k点の状態密度の上限（MaxCapacity）を満たすまで再配置。
3. **再計算:** 変更後の占有数データをWIEN2kに再投入し、励起状態下での誘電関数および吸収係数を算出。

## データ処理・可視化パイプライン (Data Pipeline)
膨大な第一原理計算の出力結果（9つの配置パターン × 複数の励起強度）を効率的に解析するため、Google Colab上で動作するデータ集計・可視化環境を構築しました。

* **自動集計:** 複数のテキストデータから必要な物理量（誘電関数の実部・虚部、吸収係数など）をパース。
* **アンサンブル平均:** 各配置パターン（`GaPN_01`~`09`）の出現確率（等価構造数）に基づく重み付き平均を自動計算。
* **プロット:** Matplotlibを用いて、光子エネルギー（eV）に対する吸収係数（$\mathrm{cm}^{-1}$）の対数グラフを自動生成。
